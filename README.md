# 📱 Ứng dụng Quản lý Công việc Cá nhân (To-do + Reminder)

## 📌 Giới thiệu

Ứng dụng giúp người dùng quản lý công việc hiệu quả với các chức năng:

- ✅ Tạo và quản lý danh sách công việc (Task)
- ⏰ Đặt deadline cho từng công việc
- 🔔 Nhắc việc (Reminder) thông minh
- ☁️ Đồng bộ dữ liệu trên nhiều thiết bị

---

## 🚀 Công nghệ sử dụng

| Thành phần  | Công nghệ |
| ----------- | --------- |
| 🎨 Frontend | Flutter   |
| 🔥 Backend  | Firebase  |

## ⚙️ Cài đặt môi trường

### 1. Cài đặt Flutter

- Tải và cài đặt tại: https://flutter.dev
- Kiểm tra cài đặt:

```bash
flutter doctor
```

---

### 2. Cài đặt dependencies

```bash
flutter pub get
```

---

## 🔥 Cấu hình Firebase

### Bước 1: Cài đặt Firebase CLI

```bash
npm install -g firebase-tools
```

### Bước 2: Đăng nhập Firebase

```bash
firebase login
```

### Bước 3: Cài đặt FlutterFire CLI

```bash
dart pub global activate flutterfire_cli
```

### Bước 4: Kết nối Firebase với project

```bash
flutterfire configure
```

---

## ▶️ Chạy ứng dụng

### Chạy trên Chrome

```bash
flutter run -d chrome
```

### Chạy trên Android / iOS

```bash
flutter run
```
